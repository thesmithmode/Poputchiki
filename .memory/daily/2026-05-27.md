**Decisions Made:**

- `postgis/postgis:16-3.4-alpine` — drop-in замена postgres образа, данные не теряются

**Lessons Learned:**
- GiST индекс не работает с `::geography` cast на geometry колонке — нужен функциональный индекс `USING GIST (geography(route_geom))`

- OSRM Docker: `osrm/osrm-backend` на DockerHub устарел → `ghcr.io/project-osrm/osrm-backend:latest`
- OSRM нет `/health` endpoint, нет curl в Alpine → healthcheck через `wget` + `/nearest/`
- `osmium` не входит в OSRM Docker образ → нужен отдельный контейнер для clip PBF
- OSRM вызов нельзя делать внутри DB транзакции — держит connection 5s при задержке сети
