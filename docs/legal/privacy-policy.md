# Политика конфиденциальности Попутчики

**Версия:** 1.0
**Дата:** 2026-05-06
**SHA256 (этого файла):** вычисляется при деплое и сохраняется в audit_log при первом принятии пользователем

---

## 1. Оператор данных

Оператор персональных данных — физическое лицо, администратор сервиса «Попутчики» (далее — «Оператор»). Контакт: через Telegram-бот или электронную почту, указанную в боте.

Сервис «Попутчики» — Telegram MiniApp для жителей ЖК Царёво (Казань), предназначенный для организации совместных поездок между соседями на безвозмездной основе.

---

## 2. Перечень обрабатываемых данных

При использовании сервиса обрабатываются следующие персональные данные:

| Данные | Источник | Обязательность |
|--------|----------|----------------|
| Telegram ID (tg_id) | Telegram | Обязательно |
| Имя (display_name) | Telegram | Обязательно |
| Имя пользователя Telegram (tg_username) | Telegram | Необязательно |
| Аватар (avatar_url) | Telegram | Необязательно |
| Номер телефона | Пользователь при онбординге | Необязательно |
| Номер квартиры | Пользователь при онбординге | Необязательно |
| Гео-точки поездок (адреса отправления и назначения) | Пользователь при создании поездки | По усмотрению |
| Время запросов и действий | Автоматически | Технически необходимо |

Номер телефона и номер квартиры хранятся в зашифрованном виде (pgcrypto, симметричное шифрование).

---

## 3. Цели обработки данных

- Предоставление функционала сервиса: публикация поездок, запись в попутчики, отзывы, лайки
- Идентификация и аутентификация пользователей через Telegram
- Обеспечение безопасности: предотвращение злоупотреблений, бан нарушителей
- Ведение журнала действий (audit log) для расследования инцидентов
- Уведомления через Telegram-бота

---

## 4. Правовое основание обработки

- Статья 6 ч. 1 п. 5 Федерального закона № 152-ФЗ «О персональных данных»: исполнение договора, стороной которого является субъект персональных данных
- Согласие субъекта персональных данных, выраженное при первом входе в сервис

---

## 5. Сроки хранения данных

| Данные | Срок хранения |
|--------|---------------|
| Аккаунт пользователя | До удаления пользователем или 3 года с последнего входа |
| Данные поездок | 1 год с даты поездки |
| Журнал действий (audit log) | 1 год |
| Токены отзыва | 60 дней |
| После анонимизации (self-delete) | Анонимизированные записи хранятся бессрочно для целостности истории поездок |

---

## 6. Передача данных третьим лицам

Персональные данные **не передаются и не продаются** третьим лицам.

Техническое исключение: передача минимально необходимых данных (Telegram ID) в Telegram API для отправки уведомлений — в соответствии с политикой конфиденциальности Telegram.

---

## 7. Право на удаление (право быть забытым)

Пользователь вправе в любой момент удалить свой аккаунт через настройки профиля. При удалении:

- Персональные данные анонимизируются (имя заменяется на «Удалённый», контактные данные обнуляются)
- Активные поездки в роли водителя отменяются
- Записи об участии в поездках **не удаляются** — они необходимы для сохранения истории отзывов других участников
- Аккаунт блокируется для повторного входа с тем же Telegram ID

Срок обработки запроса на удаление: немедленно (real-time).

---

## 8. Безопасность данных

- Хранение на выделенном сервере в России
- Шифрование чувствительных полей (телефон, квартира) на уровне базы данных
- HTTPS для всех соединений
- JWT-аутентификация с ротацией токенов
- Row-Level Security на уровне базы данных

---

## 9. Cookies и локальное хранилище

Сервис использует JWT-токены в httpOnly cookies исключительно для аутентификации. Сторонние cookies не используются. Аналитика третьих сторон не подключена.

---

## 10. Контакт оператора

По вопросам обработки персональных данных обращайтесь через Telegram-бот сервиса «Попутчики» или по электронной почте, указанной в описании бота.

---

## 11. Изменения политики

При существенном изменении настоящей политики пользователи будут уведомлены через Telegram-бот. Продолжение использования сервиса означает согласие с новой редакцией.
