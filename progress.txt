```
## [TASK-XXX] Краткое название
**Дата:** YYYY-MM-DD HH:MM (UTC)
**Статус:** done | blocked | partial

### Что сделано
- Пункт 1
- Пункт 2

- [x] Тест 2 — прошёл
- Coverage: X%

### Заметки

```

При BLOCKED — заменить раздел «Что сделано» на:
```
### BLOCKED:<краткая причина>
- Подробное описание что мешает продолжать
- Что нужно сделать вручную чтобы разблокировать
```
