# Структура репозитория

- `saas-project-ralph/` - основной проект (бот, исходный код, тесты, документация, Docker-конфигурация).
- `one-shot-по-тупому/` - отдельный one-shot эксперимент с фронтенд-проектом `billion-dollar-saas`.
- `prd/` - материалы по PRD: инструкции и рабочие заметки.
- `prd->tasks/` - артефакты, связанные с преобразованием PRD в задачи (`tasks.json`, промпты и связанные файлы).
